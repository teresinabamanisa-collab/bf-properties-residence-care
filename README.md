# BF Properties - Residence Care

Projet React + Vite prêt pour Vercel.

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer sur Vercel

1. Créer un repository GitHub.
2. Uploader tous les fichiers de ce dossier.
3. Importer le repository dans Vercel.
4. Framework preset: Vite
5. Build command: `npm run build`
6. Output directory: `dist`

## Étape suivante

Connecter Supabase pour remplacer les données fictives par de vraies données.
